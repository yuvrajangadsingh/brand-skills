# Design System: The Foundation for your Design System - shadcn/ui

> Extracted from [https://ui.shadcn.com](https://ui.shadcn.com) by brandmd

## 1. Visual Theme & Atmosphere

**Overall mood:** Balanced and professional

**Density:** Moderate. The layout uses a varied spacing scale.

**Shape language:** Rounded, friendly aesthetic with generous corner radii.

**Depth:** Uses 4 shadow styles for layering and elevation.

## 2. Color Palette & Roles

- **White** (`#FFFFFF`): Page background
- **Dark gray** (`#737373`): Secondary background
- **Black** (`#000000`): Dark background / footer
- **Orange** (`#F0B100`): Accent background
- **White** (`#FAFAFA`): Light text (on dark)
- **Light gray** (`#E5E5E5`): Divider / border
- **Black** (`#171717`): Divider / border

## 3. Typography Rules

**Primary font:** Geist

**Fonts by role:**
- Headings: Geist
- Body: Geist

**Type scale:**
- Headings: 30px, 36px, 48px
- Body / UI: 14px, 16px, 18px
- Captions / Small: 12px, 12.8px

**Weights in use:** 400, 500, 600

**Line heights:** 20px, 24px, 16px, 19.25px, 28px, 21px, 19.2px, 36px, 52.8px, 40px

**Letter spacing:** 0.6px, -2.4px, -0.45px

## 4. Component Stylings

### Buttons
- Text color: `rgb(0, 0, 0)`
- Corner radius: 8px
- Padding: 0px 0px 0px 0px
- Font: 14px, weight 500

### Cards
- Padding: 0px 0px 0px 0px

### Inputs
- Border: 0px solid rgba(0, 0, 0, 0)
- Padding: 4px 6px 4px 10px
- Font size: 14px

## 5. Layout Principles

**Spacing scale:** 4px, 6px, 8px, 10px, 12px, 14px, 16px, 20px

**Base unit:** 4px grid (values are mostly multiples of 4).

**Border radii:** 4px 4px 0px 0px, 6px 6px 0px 0px, 8px, 8px 8px 0px 0px, 18px, 19.6px, 24px, 24px 24px 0px 0px

## 6. Guidelines

### Do
- Use 4px grid for all spacing
- Use `#F0B100` for primary actions and CTAs
- Stick to 3 font weights: 400, 500, 600
- Use `Geist` as the primary typeface

### Don't
- Don't introduce colors outside the palette above
- Don't mix fonts. Use Geist everywhere
- Don't use inline styles when the design system covers the pattern
- Don't use border-radius values outside: 4px 4px 0px 0px, 6px 6px 0px 0px, 8px, 8px 8px 0px 0px, 18px, 19.6px, 24px, 24px 24px 0px 0px
